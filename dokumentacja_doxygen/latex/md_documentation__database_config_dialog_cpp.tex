Plik {\ttfamily \mbox{\hyperlink{_database_config_dialog_8cpp}{Database\+Config\+Dialog.\+cpp}}} zawiera implementację metod klasy {\ttfamily \mbox{\hyperlink{class_database_config_dialog}{Database\+Config\+Dialog}}}, odpowiedzialnej za konfigurację parametrów połączenia z bazą danych (SQLite lub My\+SQL) oraz wybór skórki graficznej aplikacji (Amiga, Atari 8bit, ZX Spectrum, Standard). \begin{DoxyEnumerate}
\item {\bfseries{Inicjalizacja}}\+:
\begin{DoxyItemize}
\item Konstruktor konfiguruje interfejs użytkownika ({\ttfamily ui-\/\texorpdfstring{$>$}{>}setup\+Ui}), wypełnia combo boxy opcjami bazy danych (SQLite3, My\+SQL) i skórek graficznych (Amiga, Atari 8bit, ZX Spectrum, Standard).
\item Wczytuje zapisane ustawienia z {\ttfamily QSettings} i ustawia początkowe wartości pól (np. ścieżka SQLite, dane My\+SQL, wybrana skórka).
\item Ustanawia połączenia sygnałów-\/slotów dla przycisków, combo boxów i wyboru pliku SQLite.
\end{DoxyItemize}
\item {\bfseries{Wybór pliku SQLite}}\+:
\begin{DoxyItemize}
\item Przycisk {\ttfamily select\+File\+Button} otwiera okno dialogowe {\ttfamily QFile\+Dialog\+::get\+Save\+File\+Name}, umożliwiając wybór istniejącego lub nowego pliku {\ttfamily .db}.
\item Jeśli wybrany plik nie istnieje, tworzony jest pusty plik.
\end{DoxyItemize}
\item {\bfseries{Zmiana typu bazy danych}}\+:
\begin{DoxyItemize}
\item Slot {\ttfamily on\+Database\+Type\+Changed} przełącza strony w {\ttfamily QStacked\+Widget}, pokazując pola odpowiednie dla SQLite lub My\+SQL.
\end{DoxyItemize}
\item {\bfseries{Zmiana skórki graficznej}}\+:
\begin{DoxyItemize}
\item Slot {\ttfamily on\+Skin\+Changed} wywołuje {\ttfamily load\+Style\+Sheet} i {\ttfamily load\+Font}, aktualizując styl i czcionkę aplikacji.
\item Wybór skórki jest zapisywany w {\ttfamily QSettings}.
\end{DoxyItemize}
\item {\bfseries{Zatwierdzanie ustawień}}\+:
\begin{DoxyItemize}
\item Metoda {\ttfamily accept} zapisuje wszystkie parametry (baza danych, skórka) w {\ttfamily QSettings} i zamyka okno dialogowe.
\end{DoxyItemize}
\item {\bfseries{Ładowanie stylów i czcionek}}\+:
\begin{DoxyItemize}
\item {\ttfamily load\+Style\+Sheet} wczytuje plik QSS z zasobów Qt i stosuje go za pomocą {\ttfamily q\+App-\/\texorpdfstring{$>$}{>}set\+Style\+Sheet}.
\item {\ttfamily load\+Font} ładuje czcionkę (np. {\ttfamily topaz.\+ttf} dla Amigi) za pomocą {\ttfamily QFont\+Database}, ustawia ją dla aplikacji i resetuje paletę/styl dla propagacji zmian.
\end{DoxyItemize}
