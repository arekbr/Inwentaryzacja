\doxysubsection{Opis szczegółowy}
Model proxy do filtrowania listy eksponatów. 

\hypertarget{class_zoomable_graphics_view_ClassOverview}{}\doxysubsection{Class\+Overview}\label{class_zoomable_graphics_view_ClassOverview}
Klasa \mbox{\hyperlink{class_item_filter_proxy_model}{Item\+Filter\+Proxy\+Model}} dziedziczy po QSort\+Filter\+Proxy\+Model i dostarcza mechanizm dynamicznego filtrowania danych eksponatów na podstawie atrybutów takich jak typ, producent, model, status i miejsce przechowywania. Umożliwia ustawienie filtrów poprzez dedykowane metody i implementuje logikę filtrowania w metodzie filter\+Accepts\+Row.\hypertarget{class_zoomable_graphics_view_Responsibilities}{}\doxysubsection{Responsibilities}\label{class_zoomable_graphics_view_Responsibilities}

\begin{DoxyItemize}
\item Ustawianie filtrów dla różnych atrybutów eksponatu.
\item Decydowanie, które wiersze modelu źródłowego spełniają kryteria filtrów.
\item Współpraca z widokami Qt (np. QTable\+View) do wyświetlania przefiltrowanych danych. 
\end{DoxyItemize}
