Plik {\ttfamily \mbox{\hyperlink{item_list_8cpp}{item\+List.\+cpp}}} zawiera implementację metod klasy {\ttfamily \mbox{\hyperlink{classitem_list}{item\+List}}}, która zarządza listą eksponatów w aplikacji inwentaryzacyjnej. Obsługuje połączenie z bazą danych My\+SQL, filtrowanie kaskadowe, wyświetlanie miniatur zdjęć, podgląd pełnoekranowy oraz operacje CRUD (dodawanie, edycja, usuwanie, klonowanie rekordów). Skórka graficzna i czcionki są zarządzane przez klasę {\ttfamily \mbox{\hyperlink{class_database_config_dialog}{Database\+Config\+Dialog}}}.\hypertarget{md_documentation_item_list_cpp_autotoc_md52}{}\doxysection{Struktura kodu}\label{md_documentation_item_list_cpp_autotoc_md52}

\item {\bfseries{Inicjalizacja}}\+:
\begin{DoxyItemize}
\item Konstruktor ustawia interfejs użytkownika ({\ttfamily ui-\/\texorpdfstring{$>$}{>}setup\+Ui}), łączy się z bazą My\+SQL, inicjalizuje model danych ({\ttfamily QSql\+Relational\+Table\+Model}) z relacjami do tabel {\ttfamily types}, {\ttfamily vendors}, {\ttfamily models}, {\ttfamily statuses}, {\ttfamily storage\+\_\+places}.
\item Konfiguruje model proxy ({\ttfamily \mbox{\hyperlink{class_item_filter_proxy_model}{Item\+Filter\+Proxy\+Model}}}), tabelę ({\ttfamily QTable\+View}) i combo boxy dla filtrów.
\item Weryfikuje schemat bazy danych, tworząc go i wstawiając przykładowe dane, jeśli nie istnieje.
\item Ustawia połączenia sygnałów-\/slotów dla przycisków, tabeli i filtrów.
\end{DoxyItemize}
\item {\bfseries{Filtrowanie}}\+:
\begin{DoxyItemize}
\item Metoda {\ttfamily on\+Filter\+Changed} aktualizuje filtry w modelu proxy na podstawie wybranych wartości w combo boxach.
\item {\ttfamily update\+Filter\+Combo\+Boxes} odbudowuje listy combo boxów z uwzględnieniem kaskadowego filtrowania, pobierając unikalne wartości z bazy danych.
\item {\ttfamily init\+Filters} i {\ttfamily refresh\+Filters} wypełniają i odświeżają combo boxy danymi z bazy.
\end{DoxyItemize}
\item {\bfseries{Operacje CRUD}}\+:
\begin{DoxyItemize}
\item {\ttfamily on\+New\+Button\+Clicked}, {\ttfamily on\+Edit\+Button\+Clicked}, {\ttfamily on\+Clone\+Button\+Clicked}\+: Otwierają okno {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} w odpowiednim trybie (dodawanie, edycja, klonowanie).
\item {\ttfamily on\+Delete\+Button\+Clicked}\+: Usuwa rekord po potwierdzeniu, wykonując zapytanie SQL i odświeżając model.
\item {\ttfamily on\+Record\+Saved}\+: Odświeża listę po zapisaniu rekordu, zaznaczając zapisany rekord.
\end{DoxyItemize}
\item {\bfseries{Zdjęcia}}\+:
\begin{DoxyItemize}
\item {\ttfamily on\+Table\+View\+Selection\+Changed}\+: Pobiera zdjęcia z tabeli {\ttfamily photos} dla wybranego eksponatu i wyświetla miniatury w {\ttfamily QGraphics\+View}.
\item {\ttfamily on\+Photo\+Hovered}\+: Pokazuje powiększone zdjęcie w oknie podglądu, pozycjonowanym względem miniatury.
\item {\ttfamily on\+Photo\+Unhovered} i {\ttfamily event\+Filter}\+: Zamykają okno podglądu, gdy kursor opuszcza miniaturę lub okno.
\item {\ttfamily on\+Photo\+Clicked}\+: Otwiera zdjęcie w trybie pełnoekranowym za pomocą {\ttfamily \mbox{\hyperlink{class_full_screen_photo_viewer}{Full\+Screen\+Photo\+Viewer}}}.
