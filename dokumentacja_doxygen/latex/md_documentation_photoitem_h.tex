Plik {\ttfamily \mbox{\hyperlink{photoitem_8h}{photoitem.\+h}}} definiuje klasę {\ttfamily \mbox{\hyperlink{class_photo_item}{Photo\+Item}}}, która reprezentuje interaktywny element graficzny (miniaturę zdjęcia) w scenie graficznej Qt (używany w {\ttfamily QGraphics\+View}). Klasa rozszerza {\ttfamily QGraphics\+Pixmap\+Item} o obsługę zdarzeń myszy (kliknięcia, podwójne kliknięcia) i hover (najechanie, opuszczenie kursora), a także wizualne zaznaczanie za pomocą czerwonej ramki. Dzięki dziedziczeniu po {\ttfamily QObject}, umożliwia emisję sygnałów ({\ttfamily clicked}, {\ttfamily double\+Clicked}, {\ttfamily hovered}, {\ttfamily unhovered}), co pozwala na integrację z logiką aplikacji, np. z klasą {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} w aplikacji inwentaryzacyjnej.\hypertarget{md_documentation_photoitem_h_autotoc_md109}{}\doxysection{Struktura kodu}\label{md_documentation_photoitem_h_autotoc_md109}

\begin{DoxyItemize}
\item {\bfseries{Inicjalizacja}}\+: Konstruktor tworzy obiekt jako element graficzny ({\ttfamily QGraphics\+Pixmap\+Item}) i obiekt sygnałowy ({\ttfamily QObject}), ustawia flagi {\ttfamily m\+\_\+pressed} i {\ttfamily m\+\_\+selected} na {\ttfamily false}, tworzy ramkę ({\ttfamily m\+\_\+frame}) jako dziecko, włącza zdarzenia hover i konfiguruje ramkę jako niewidoczną.
\item {\bfseries{Interfejs graficzny}}\+: Wyświetla miniaturę zdjęcia (np. 80x80 pikseli w {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}}) w {\ttfamily QGraphics\+View}, z możliwością zaznaczenia czerwoną ramką o szerokości 2 piksele.
\item {\bfseries{Interakcje}}\+:
\begin{DoxyItemize}
\item {\bfseries{Kliknięcia}}\+: Prawe kliknięcie emituje {\ttfamily clicked} (np. dla podglądu), lewe jest przekazywane dla podwójnego kliknięcia.
\item {\bfseries{Podwójne kliknięcia}}\+: Lewe podwójne kliknięcie emituje {\ttfamily double\+Clicked} (np. dla pełnego ekranu).
\item {\bfseries{Hover}}\+: Najechanie emituje {\ttfamily hovered}, opuszczenie {\ttfamily unhovered} (np. dla podświetlenia).
\end{DoxyItemize}
\item {\bfseries{Zaznaczenie}}\+: Metoda {\ttfamily set\+Selected} włącza/wyłącza czerwoną ramkę poprzez {\ttfamily update\+Frame}, co jest używane w {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} do wskazywania wybranego zdjęcia.
\item {\bfseries{Integracja}}\+: Sygnały są podłączane w {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} do slotów, np. {\ttfamily on\+Photo\+Clicked}, dla wyboru lub podglądu zdjęcia.
