Plik {\ttfamily \mbox{\hyperlink{_item_filter_proxy_model_8cpp}{Item\+Filter\+Proxy\+Model.\+cpp}}} zawiera implementację metod klasy {\ttfamily \mbox{\hyperlink{class_item_filter_proxy_model}{Item\+Filter\+Proxy\+Model}}}, która umożliwia dynamiczne filtrowanie listy eksponatów w aplikacji inwentaryzacyjnej na podstawie atrybutów takich jak typ, producent, model, status i miejsce przechowywania. \end{DoxyItemize}
\item {\bfseries{Filtrowanie}}\+:
\begin{DoxyItemize}
\item Metoda {\ttfamily filter\+Accepts\+Row} sprawdza, czy wartości w kolumnach modelu źródłowego (2\+: typ, 3\+: producent, 4\+: model, 9\+: status, 10\+: miejsce przechowywania) odpowiadają ustawionym filtrom.
\item Używa lambdy {\ttfamily pasuje}, która porównuje wartość w kolumnie z wartością filtru. Jeśli filtr jest pusty, wszystkie wartości dla tej kolumny są akceptowane.
\item Wiersz jest widoczny tylko wtedy, gdy spełnia wszystkie aktywne filtry.
