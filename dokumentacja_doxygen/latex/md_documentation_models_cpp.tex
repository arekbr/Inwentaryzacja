Plik {\ttfamily \mbox{\hyperlink{models_8cpp}{models.\+cpp}}} zawiera implementację metod klasy {\ttfamily models}, odpowiedzialnej za zarządzanie modelami eksponatów w aplikacji inwentaryzacyjnej. Klasa umożliwia dodawanie, edytowanie i usuwanie modeli w tabeli {\ttfamily models} w bazie danych My\+SQL, współpracując z interfejsem użytkownika ({\ttfamily QList\+View}, {\ttfamily QLine\+Edit}) oraz klasą {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} w celu odświeżania combo boxów. Obsługuje powiązanie modeli z producentami poprzez {\ttfamily vendor\+Id} (QString).\hypertarget{md_documentation_models_cpp_autotoc_md88}{}\doxysection{Struktura kodu}\label{md_documentation_models_cpp_autotoc_md88}

\item {\bfseries{Inicjalizacja}}\+:
\begin{DoxyItemize}
\item Konstruktor ustawia interfejs ({\ttfamily ui-\/\texorpdfstring{$>$}{>}setup\+Ui}), pobiera połączenie z bazą My\+SQL ({\ttfamily default\+\_\+connection}), ustala tytuł okna, podłącza sloty dla przycisków (Dodaj, Edytuj, Usuń, OK, Anuluj) i wywołuje {\ttfamily refresh\+List} do wyświetlenia modeli.
\item Inicjalizuje {\ttfamily m\+\_\+vendor\+Id} jako pusty QString i {\ttfamily m\+\_\+main\+Window} jako nullptr.
\end{DoxyItemize}
\item {\bfseries{Arithmetic\+Error}}\+:
\begin{DoxyItemize}
\item Zawiera {\ttfamily QList\+View} do wyświetlania modeli (sortowanych alfabetycznie), {\ttfamily QLine\+Edit} do wprowadzania nazwy modelu oraz przyciski do zarządzania modelami i zamykania okna.
\item {\ttfamily QSql\+Query\+Model} dostarcza dane do {\ttfamily QList\+View}.
\end{DoxyItemize}
\item {\bfseries{Zarządzanie modelami}}\+:
\begin{DoxyItemize}
\item {\bfseries{Dodawanie}} ({\ttfamily on\+Add\+Clicked})\+:
\begin{DoxyItemize}
\item Pobiera nazwę z {\ttfamily QLine\+Edit}, waliduje (niepusta).
\item Generuje UUID dla modelu.
\item Pobiera {\ttfamily vendor\+Id} z {\ttfamily m\+\_\+vendor\+Id} lub combo boxa {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} (domyślnie {\ttfamily unknown\+\_\+vendor\+\_\+uuid}).

\item {\bfseries{Edycja}} ({\ttfamily on\+Edit\+Clicked})\+:
\begin{DoxyItemize}
\item Sprawdza, czy wybrano model w {\ttfamily QList\+View}.
\item Otwiera {\ttfamily QInput\+Dialog} z aktualną nazwą, waliduje nową nazwę.
\item Pobiera ID modelu po nazwie, wykonuje {\ttfamily UPDATE} i odświeża listę.
\end{DoxyItemize}
\item {\bfseries{Usuwanie}} ({\ttfamily on\+Delete\+Clicked})\+:
\begin{DoxyItemize}
\item Sprawdza, czy wybrano model.
\item Prosi o potwierdzenie w {\ttfamily QMessage\+Box}.
\item Wykonuje {\ttfamily DELETE} dla nazwy modelu i odświeża listę.
\end{DoxyItemize}
\item Wyświetla komunikaty o błędach SQL lub brakujących wyborach.

\end{DoxyItemize}
\item {\bfseries{Filtr producentów}}\+:
