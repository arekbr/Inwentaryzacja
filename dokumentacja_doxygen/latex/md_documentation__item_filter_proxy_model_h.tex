Plik {\ttfamily \mbox{\hyperlink{_item_filter_proxy_model_8h}{Item\+Filter\+Proxy\+Model.\+h}}} definiuje klasę {\ttfamily \mbox{\hyperlink{class_item_filter_proxy_model}{Item\+Filter\+Proxy\+Model}}}, która dziedziczy po {\ttfamily QSort\+Filter\+Proxy\+Model} i służy do dynamicznego filtrowania listy eksponatów w aplikacji inwentaryzacyjnej. Klasa umożliwia ustawienie filtrów dla atrybutów eksponatu, takich jak typ, producent, model, status i miejsce przechowywania, oraz kontroluje, które wiersze modelu źródłowego są wyświetlane w widoku na podstawie zadanych kryteriów.\hypertarget{md_documentation__item_filter_proxy_model_h_autotoc_md45}{}\doxysection{Struktura kodu}\label{md_documentation__item_filter_proxy_model_h_autotoc_md45}
