Plik {\ttfamily \mbox{\hyperlink{storage_8cpp}{storage.\+cpp}}} zawiera implementację metod klasy {\ttfamily storage}, odpowiedzialnej za zarządzanie miejscami przechowywania eksponatów (np. magazyny, półki, pokoje) w aplikacji inwentaryzacyjnej. Klasa umożliwia dodawanie, edytowanie i usuwanie lokalizacji w tabeli {\ttfamily storage\+\_\+places} w bazie danych My\+SQL, współpracując z interfejsem użytkownika ({\ttfamily QList\+View}, {\ttfamily QLine\+Edit}) oraz klasą {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} w celu odświeżania combo boxów lokalizacji po zapisaniu zmian.\hypertarget{md_documentation_storage_cpp_autotoc_md130}{}\doxysection{Struktura kodu}\label{md_documentation_storage_cpp_autotoc_md130}

\begin{DoxyEnumerate}
\item {\bfseries{Inicjalizacja}}\+:
\begin{DoxyItemize}
\item Konstruktor ustawia interfejs ({\ttfamily ui-\/\texorpdfstring{$>$}{>}setup\+Ui}), pobiera połączenie z bazą My\+SQL ({\ttfamily default\+\_\+connection}), ustala tytuł okna, podłącza sloty dla przycisków (Dodaj, Edytuj, Usuń, OK, Anuluj) i wywołuje {\ttfamily refresh\+List} do wyświetlenia lokalizacji.
\item Inicjalizuje {\ttfamily m\+\_\+main\+Window} jako {\ttfamily nullptr}.
\end{DoxyItemize}
\item {\bfseries{Interfejs użytkownika}}\+:
\begin{DoxyItemize}
\item Zawiera {\ttfamily QList\+View} do wyświetlania lokalizacji (sortowanych alfabetycznie), {\ttfamily QLine\+Edit} do wprowadzania nazwy lokalizacji oraz przyciski do zarządzania lokalizacjami i zamykania okna.
\item {\ttfamily QSql\+Query\+Model} dostarcza dane do {\ttfamily QList\+View}.
\end{DoxyItemize}
\item {\bfseries{Zarządzanie lokalizacjami}}\+:
\begin{DoxyItemize}
\item {\bfseries{Dodawanie}} ({\ttfamily on\+Add\+Clicked})\+:
\begin{DoxyItemize}
\item Pobiera nazwę z {\ttfamily QLine\+Edit}, waliduje (niepusta).
\item Generuje UUID dla lokalizacji.
\item Wykonuje {\ttfamily INSERT} do tabeli {\ttfamily storage\+\_\+places}, odświeża listę i czyści pole.
\item {\bfseries{Błąd}}\+: Linia {\ttfamily QUuid\+::create\+Uuid().to\+String(\+QUuid\+::\+Without\+Braces);} generuje UUID, ale nie przypisuje go (wynik jest odrzucany).
\end{DoxyItemize}
\item {\bfseries{Edycja}} ({\ttfamily on\+Edit\+Clicked})\+:
\begin{DoxyItemize}
\item Sprawdza, czy wybrano lokalizację w {\ttfamily QList\+View}.
\item Otwiera {\ttfamily QInput\+Dialog} z aktualną nazwą, waliduje nową nazwę.
\item Pobiera ID lokalizacji po nazwie, wykonuje {\ttfamily UPDATE} i odświeża listę.
\end{DoxyItemize}
\item {\bfseries{Usuwanie}} ({\ttfamily on\+Delete\+Clicked})\+:
\begin{DoxyItemize}
\item Sprawdza, czy wybrano lokalizację.
\item Prosi o potwierdzenie w {\ttfamily QMessage\+Box}.
\item Wykonuje {\ttfamily DELETE} dla nazwy lokalizacji i odświeża listę.
\end{DoxyItemize}
\item Wyświetla komunikaty o błędach SQL lub brakujących wyborach.
\end{DoxyItemize}
\item {\bfseries{Integracja z {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}}}}\+:
\begin{DoxyItemize}
\item {\ttfamily on\+Ok\+Clicked} wywołuje {\ttfamily load\+Combo\+Box\+Data} na combo boxie lokalizacji w {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} (jeśli ustawione) i zamyka okno z wynikiem {\ttfamily accept()}.
\item {\ttfamily set\+Main\+Window} przechowuje referencję do {\ttfamily \mbox{\hyperlink{class_main_window}{Main\+Window}}} dla odświeżania.
\end{DoxyItemize}
